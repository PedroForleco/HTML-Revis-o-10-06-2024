Arduino Aprendemos de maneira eficiente e prática sobre a montagem das peças, e todo o esquema e programação por trás do funcionamento de todo o sistema, saída e entrada de energia e outros fatores básicos necessários para seu funcionamento. Tivemos um pouco de dificuldade com as variações de programações em relação aos diferentes componentes, que precisamos utilizar ao decorrer das atividades propostas.
HTML Utilizamos o HTML para o início dos sites de maneira mais aprofundada, utilizando o método de login e senha para acessar o site, criação de várias páginas, diretórios, redirecionamento de páginas através de fotos, links, botões e etc. Utilizado na maioria das vezes com o JavaScript para a criação e edição de páginas de sites. Durante a montagem de sites e porjetos, surgiu determinadas dificuldades em relação aos comandos que utilizavamos durante a construção e estruturação das ativiades propostas. Ou seja, alguma letra que digitavamos errado o projeto não ocorreia conforme o planejamneto.
Java Script Através do JavaScript vimos que é uma das maiores linguagem de programação mais utilizadas se não for a maior, sendo utilizada principalmente em sites dinâmicos para uma melhor navegação, sendo utilizado na maioria das vezes em conjunto com o HTML para suas montagens. Surgiu bastantes dificuldades durante a utilização do Java Script em nosos porjetos porque era uma lingaguem de programação relacionada a HTML.
Visual G O visual G foi de grande importância para o início da nossa programação, sendo muito versátil já que sua linguagem de código é em português, sendo mais fácil para pessoas que não tem um grande conhecimento na língua inglesa. Foi crucial para o conhecimento dos comandos básicos com “IF”, formação de expressões, cálculos de porcentagens através de códigos. A principal dificuldade nossa foi a utilização do SE e do SENÃO, para fazermos e executarmos o sistema.
Node Red Através do Node-Red aprendemos a montagem das placas arduinos físicas para executar os programas, como por exemplo exibirmos letras em leds, a sistema funcional de uma semáforo, com todos os intervalos das luzes para a liberação e fechamento das vias, juntamente intercalado com um semáforo para pedestres com o mesmo objetivo, ou seja, aprendemos basicamente d e maneira simples como montar uma hardware funcional. O principal ponto negativo nosso foi no momento da montagem do hardware, como não estamos acostumados com a montagem dessas peças, sentimos um pouco de dificuldade, nos sentimos um pouco perdidos já que não estávamos montando e quando em uma área que éramos muito especializados, mas nada que não conseguimos nos acostumar e nos adaptar após algumas aulas.
Figma Através do Figma aprendemos facilmente sobre o design de sites e aplicativos utilizados nos eletrônicos da Aplle, Iphone, Ipad, MacBooks, por meio de botões e várias imagens utilizadas em camadas e redirecionamento, sendo possível criar todo um sistema de navegação pré-definida através do aplicativo. Nossa principal dificuldade durante a realização do Figam, para fazer os sites foi a utilização dos botões para trocar as páginas dos nossos porjetos que estavamos fazendo.
